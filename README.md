# 🛒 E-Commerce Website (MERN Stack)

A full-stack **E-Commerce Web Application** built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)**.  
This project includes a **customer storefront** and a **dedicated admin portal** for managing products, orders, and users.

---

## 🚀 Features

### 🧑‍💻 User Features
- 🔐 User Authentication & Authorization (JWT)
- 🛍️ Product Listing & Filtering
- 📄 Product Details Page
- 🛒 Add to Cart & Cart Management
- 💳 Checkout Process
- 📦 My Orders Page
- 🔎 Search Products
- 📱 Fully Responsive UI

### 🛠️ Admin Portal
- ➕ Add New Products
- 📝 Edit / Delete Products
- 📦 Manage Orders (update status)
- 👥 Manage Users & Roles
- 📊 Dashboard Overview
- 🔐 Admin Protected Routes

---

## 🧰 Tech Stack

### Frontend
- React.js  
- Axios  
- Redux / Context API  
- React Router  
- Tailwind / Bootstrap / CSS  

### Backend
- Node.js  
- Express.js  
- MongoDB + Mongoose  
- JWT Authentication  
- Multer / Cloudinary for image upload
